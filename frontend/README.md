Dentro da pasta do projeto, execute o comando abaixo. Ele irá ler o arquivo package.json e baixar todas as bibliotecas necessárias para o projeto funcionar.

```javascript

npm install

```

Dentro da pasta do projeto, execute o comando abaixo. Ele irá ler o arquivo package.json e baixar a biblioteca responsável pelo calendário de agendamentos.

```javascript

npm install fullcalendar

```

### Este comando instalará:

- react e react-dom (a base da nossa aplicação)

- react-router-dom (para o sistema de rotas)

- tailwindcss (para a estilização)

- E outras dependências de desenvolvimento, como o vite.


### Dê Run

```javascript

npm run dev

```
